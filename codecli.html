Why Sahya Code?

🚀

Lightning Fast

Built for speed. Sahya Code uses efficient streaming and local caching to deliver AI responses instantly. No more waiting around.

🔒

Privacy First

Your code stays local. We prioritize privacy with on-device processing where possible and secure connections when needed.

⚡

Multi-Model Support

Use your preferred AI model. Compatible with OpenAI, Anthropic, Ollama local models, and more. Switch models on the fly.

sahya config model gpt-4
🛠️

Extensible Tools

Built-in tools for file operations, web search, code execution, and more. Create custom tools with the plugin system.

How to Use Sahya Code

$ sahyacode
Welcome to Sahya Code v2.0.4
Type your message or /help for commands
>

Interactive Mode

Run 'sahyacode' in your project folder to launch the interactive terminal UI. Use '@' to reference local files and get contextual assistance.

sahyacode
$ sahyacode -p "Explain @main.py"
This Python script contains...
1. A FastAPI application setup
2. Route definitions for /api/*
3. Database connection handling

Headless Mode

Use '-p' flag to run Sahya Code without the interactive UI — ideal for scripts, automation, and CI/CD pipelines.

sahyacode -p "Explain the auth flow"
$ sahyacode providers
Available providers:
✓ nexiant (default)
✓ openai
✓ anthropic

Easy Configuration

Configure AI providers and preferences with simple commands. Set up Nexiant, OpenAI, or your own LiteLLM instance.

sahyacode providers list

Project Architecture

🏗️ Monorepo Structure

Sahya Code is organized as a monorepo with multiple packages:

packages/opencode/

Core CLI application built with Bun and compiled to native binaries. Supports macOS, Linux (x64/ARM64), and Windows.

Key Components

  • CLI — Terminal UI built with Ink/React for interactive sessions
  • Provider Layer — Abstraction for 75+ AI providers via LiteLLM
  • MCP Integration — Model Context Protocol for tool servers
  • Session Manager — SQLite-based conversation persistence
  • File System — Project indexing and context management

🔌 Provider System

Sahya Code supports multiple AI providers through a unified interface:

Nexiant (Default)

LiteLLM endpoint at llm.nexiant.ai

OpenAI

GPT-4, GPT-3.5, and more

Anthropic

Claude 3.5 Sonnet, Opus

Ollama

Local models (Llama, etc.)

sahyacode providers list

⚙️ Connecting to LiteLLM / Nexiant

To use your own LiteLLM instance or connect to Nexiant AI:

Option 1: Interactive Setup

$ sahyacode providers add nexiant
Enter API key: ••••••••
✓ Provider configured successfully

Option 2: Environment Variable

export NEXIANT_API_KEY=your-api-key

Option 3: Custom LiteLLM URL

For custom LiteLLM deployments, add a custom provider:

$ sahyacode providers add custom
Name: my-litellm
Base URL: https://llm.nexiant.ai/v1
API Key: ••••••••

🎨 Theme Manager

Create, edit, and manage custom terminal themes with the interactive theme manager. Supports hex, RGB, and RGBA color formats with live preview.

Create a New Theme

$ sahyacode theme create
Theme name: my-awesome-theme
Primary - Main brand color
Dark mode color: #ff4f00
Light mode color: #ff6b35
✓ Theme saved!

Theme Commands

sahyacode theme create — Create a new theme
sahyacode theme edit — Edit existing theme
sahyacode theme list — List all themes
sahyacode theme export — Export theme to file
sahyacode theme import — Import theme from file

Color Input Formats

  • Hex: #ff4f00 or ff4f00
  • RGB: rgb(255, 79, 0)
  • RGBA: rgba(255, 79, 0, 0.8)

Themes are saved to ~/.config/sahyacode/themes/ (global) or .sahyacode/themes/ (project-local).
