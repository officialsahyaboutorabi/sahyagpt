Code with Sahya Code

Sahya Code is an open-source AI agent for the terminal. It helps you understand large codebases, automate tedious work, and ship faster.

>_ curl -fsSL https://sbgpt.qzz.io/install.sh | bash

Why Sahya Code?

🚀

Lightning Fast

Built for speed. Sahya Code uses efficient streaming and local caching to deliver AI responses instantly. No more waiting around.

🔒

Privacy First

Your code stays local. We prioritize privacy with on-device processing where possible and secure connections when needed.

⚡

Multi-Model Support

Use your preferred AI model. Compatible with OpenAI, Anthropic, Ollama local models, and more. Switch models on the fly.

sahya config model gpt-4
🛠️

Extensible Tools

Built-in tools for file operations, web search, code execution, and more. Create custom tools with the plugin system.

How to Use Sahya Code

$ sahya
Welcome to Sahya Code v1.0.0
Type your message or /help for commands
>

Interactive Mode

Run 'sahya' in your project folder to launch the interactive terminal UI. Use '@' to reference local files and get contextual assistance.

sahya
$ sahya -p "Explain @main.py"
This Python script contains...
1. A FastAPI application setup
2. Route definitions for /api/*
3. Database connection handling

Headless Mode

Use '-p' flag to run Sahya Code without the interactive UI — ideal for scripts, automation, and CI/CD pipelines.

sahya -p "Explain the auth flow"
$ sahya config
Current configuration:
model: gpt-4
theme: dark
auto_save: true

Easy Configuration

Configure your preferences with simple commands. Set default models, themes, and behaviors to match your workflow.

sahya config --list
