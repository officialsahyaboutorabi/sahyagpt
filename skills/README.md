# BB_Skills

A collection of AI skills for enhanced productivity and development workflows.

## Available Skills

### Superpowers Skills
- **Brainstorming**: Creative ideation and concept development
- **Subagent-driven Development**: Distributed task management
- **Test-driven Development**: Automated testing workflows
- **Writing Skills**: Content creation and documentation
- **Using Superpowers**: Advanced AI capabilities

### Product Management Skills
- **Brainstorm Ideas**: Structured ideation techniques
- **Create PRD**: Product Requirements Document generation

### Development & Code Quality Skills
- **UI Developer**: Composite skill for user interface creation and revision (combines system design, impeccable design, React best practices, TDD, and subagent-driven development)
- **Code Review**: Automated code review with multiple specialized agents
- **Security Guidance**: Security analysis and vulnerability identification
- **Commit Commands**: Git commit message generation and version control assistance

### Design & Frontend Skills
- **Impeccable Design**: Comprehensive design system with 17 specialized commands (audit, critique, normalize, polish, distill, clarify, optimize, harden, animate, colorize, bolder, quieter, delight, extract, adapt, onboard)

### Self-Improvement Skills
- **Self-Improving Agent**: Captures learnings, errors, and corrections to enable continuous improvement

### Security & Protection Skills
- **Safe Encryption**: Secure encryption methods for sensitive data with post-quantum support
- **Threat Hunting**: Proactive threat detection using sigma rules and behavioral analysis
- **Computer Forensics**: Digital forensics analysis and investigation capabilities

### AI/ML Security Skills
- **Secure Data Handling**: Secure handling of sensitive data in AI/ML applications to prevent information leakage
- **Information Leakage Prevention**: Prevents information leakage in AI/ML applications through secure practices and monitoring
- **Secure Model Deployment**: Secure deployment of AI/ML models with protection against information leakage and unauthorized access

## Usage

These skills are designed to work with AI assistants that support skill-based interactions. Each skill provides structured guidance for specific tasks.
