# Agency Agents Skill

A comprehensive collection of 144+ specialized AI agents organized across 12 business divisions. Each agent is a production-ready persona with expertise, workflows, and deliverables.

## Overview

This skill provides access to a complete AI agency - from frontend developers to Reddit community builders, from security engineers to TikTok strategists. Each agent has:

- **🎭 Strong Personality**: Unique voice, communication style, and approach
- **📋 Clear Deliverables**: Real code, processes, and measurable outcomes
- **✅ Success Metrics**: Measurable results and quality standards
- **🔄 Proven Workflows**: Step-by-step methodologies that work

## When to Use This Skill

Use this skill when you need:
- Specialized expertise for specific tasks
- Multi-agent teams for complex projects
- Domain-specific guidance (engineering, marketing, sales, etc.)
- Structured workflows and deliverables

**Trigger phrases:**
- "Activate [agent name]"
- "Hire a [specialty]"
- "Build a team for [project]"
- "Show me [division] agents"
- "Assemble [type] team"

## Agent Structure

Each agent includes:

1. **Frontmatter** - Name, description, emoji, color, vibe
2. **Identity & Memory** - Personality and role definition
3. **Core Mission** - 4-6 key responsibility areas
4. **Critical Rules** - Non-negotiable guidelines
5. **Technical Deliverables** - Code examples and templates
6. **Workflow Process** - Step-by-step methodology
7. **Communication Style** - Voice and tone guidelines
8. **Success Metrics** - Measurable outcomes
9. **Advanced Capabilities** - Expert-level features
