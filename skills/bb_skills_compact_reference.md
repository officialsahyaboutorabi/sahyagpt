## Design & Frontend Skills
- **Frontend Design**: Create distinctive, production-grade interfaces avoiding generic AI aesthetics
- **Impeccable Design**: Comprehensive design system with 17 specialized commands:
  - **audit**: Perform comprehensive interface quality audits across accessibility, performance, theming, responsive design
  - **critique**: UX design review focusing on hierarchy, clarity, emotional resonance
  - **normalize**: Align interface with design system standards and consistent patterns
  - **polish**: Apply final refinements and enhancements for production-ready quality
  - **distill**: Strip to essential elements, removing complexity and focusing on core functionality
  - **clarify**: Improve unclear UX copy, confusing instructions, ambiguous labels
  - **optimize**: Identify and implement performance improvements across loading speed, rendering efficiency
  - **harden**: Strengthen resilience with robust error handling, i18n support, edge case management
  - **animate**: Add purposeful motion and animation to enhance user experience
  - **colorize**: Introduce strategic color for visual hierarchy, brand identity, accessibility
  - **bolder**: Amplify boring designs to create more impactful, memorable interfaces
  - **quieter**: Tone down overly bold designs for more refined, subtle interfaces
  - **delight**: Add moments of joy, surprise, and positive emotion to user experiences
  - **extract**: Identify and extract reusable components, patterns, and modules
  - **adapt**: Modify interface for different devices, contexts, user needs, environmental factors
  - **onboard**: Design effective onboarding flows that guide new users and create positive impressions
